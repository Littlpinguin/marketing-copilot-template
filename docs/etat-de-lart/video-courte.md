# État de l'art — Vidéo courte 2026 (TikTok / Reels / Shorts / LinkedIn)

> Recherche multi-sources, juillet 2026. Règles actionnables pour un agent IA qui écrit des scripts, monte ou brief des vidéos courtes. Le paysage a changé : l'ère du « toujours plus court » est finie — c'est le **taux de complétion** qui gouverne la distribution, pas la durée brute.

---

## 1. Règles vérifiées

### A. Hook et rétention (transversal à toutes les plateformes)

**R1. 50-60 % des abandons ont lieu dans les 3 premières secondes. Le hook doit être visuel + verbal + textuel simultanément dès la frame 1 — jamais de logo, jingle ou plan d'installation en ouverture.**
Concrètement pour un script : la première phrase est la promesse ou la tension (« Voici pourquoi X », résultat montré avant le process, question qui pique), le premier plan est déjà l'action, et le texte à l'écran reformule le hook.
Sources : [OpusClip — YouTube Shorts retention data](https://www.opus.pro/blog/ideal-youtube-shorts-length-format-retention) ; [Socialync — Short-form video structure 2026](https://www.socialync.io/blog/short-form-video-structure-guide-2026) ; [Teleprompter — Short-form strategy 2026](https://www.teleprompter.com/blog/short-form-video-strategy).

**R2. Structure validée : Hook (0-3 s) → Value drop (4-15 s) → Payoff/révélation (16-45 s) → CTA (5 dernières secondes). Un nouveau beat visuel ou narratif toutes les 5-7 secondes.**
Le payoff est la raison pour laquelle le spectateur est resté : réponse, reveal, punchline, transformation. Sans payoff explicite prévu au script, ne pas produire la vidéo. Les vidéos « bouclables » (fin qui renvoie au début) sont favorisées car les replays comptent dans la complétion.
Sources : [Socialync — structure guide 2026](https://www.socialync.io/blog/short-form-video-structure-guide-2026) ; [Real Marketing Solutions — video structure](https://realmarketingsolutions.net/best-structure-for-short-form-videos/) ; [VersaCreative — Shorts algorithm 2025](https://versacreative.com/blog/how-the-youtube-shorts-algorithm-works-in-2025/).

**R3. La complétion bat la durée : une vidéo de 45 s complétée à 70 % surperforme une vidéo de 15 s complétée à 40 %. Choisir la durée en fonction de la matière, puis couper tout ce qui n'est pas indispensable.**
Sous ~90 s, la rétention moyenne reste ≈ 50 % ; au-delà, la complétion chute nettement. Règle de production : écrire le script, puis le raccourcir de 20-30 % au montage (supprimer intros, transitions, redites).
Sources : [Shortimize — Video length sweet spots 2025](https://www.shortimize.com/blog/video-length-sweet-spots-tiktok-reels-shorts) ; [Digital Applied — Shorts vs TikTok vs Reels](https://www.digitalapplied.com/blog/short-form-video-strategy-shorts-tiktok-reels-2026).

### B. Sound-off et sous-titres

**R4. Sous-titres incrustés obligatoires sur 100 % des vidéos : ~85 % des vidéos social sont regardées sans le son (jusqu'à 92 % sur mobile), et jusqu'à 80 % des spectateurs finissent plus volontiers une vidéo sous-titrée (Verizon/Publicis Media).**
La vidéo doit être **compréhensible en muet de bout en bout** : sous-titres synchronisés mot à mot ou par groupe court, texte d'écran pour les infos clés, et ne jamais porter une information uniquement par la bande-son. 69 % coupent le son en public ; ~70 % des Américains regardent avec sous-titres par défaut (59 % de la Gen Z les active même dans sa langue).
Sources : [Kapwing — Subtitle statistics](https://www.kapwing.com/resources/subtitle-statistics/) ; [Verbit — Verizon/Publicis captions study](https://verbit.ai/captioning/sound-on-sound-off-64-marketers-using-video-see-benefit-of-captions/) ; [Sonix — Subtitle trends 2026](https://sonix.ai/resources/subtitle-generation-trends/).

**R5. Sous-titres : gros, centrés dans la zone sûre (éviter le tiers bas recouvert par l'UI), 1-2 lignes max, mots-clés mis en évidence. Prévoir quand même un bon son (musique tendance sur TikTok/Reels = signal de distribution).**
Sources : [Veed — Why subtitles 2025](https://veed.io/learn/why-adding-subtitles-to-your-video-is-a-must-in-2021) ; [Metricool — TikTok video length 2025](https://metricool.com/tiktok-video-length/).

### C. Durées optimales par plateforme (2025-2026)

**R6. Adapter la durée à la plateforme — le « 21-34 s universel » est obsolète :**
- **TikTok** : le contenu à substance performe désormais sur 60-180 s (l'algorithme valorise le watch time cumulé) ; divertissement pur : 18-25 s.
- **Instagram Reels** : 15-30 s pour la portée virale ; 60-90 s pour le storytelling et l'engagement qualifié.
- **YouTube Shorts** : 25-40 s = meilleure zone de watch-through ; l'éducatif tient 35-45 s si le payoff est clair.
- **LinkedIn vidéo** : < 60-90 s, verticale, intervenant visible et insight dans les 3 premières secondes.
Sources : [Shortimize — sweet spots 2025](https://www.shortimize.com/blog/video-length-sweet-spots-tiktok-reels-shorts) ; [OpusClip — TikTok retention data](https://www.opus.pro/blog/tiktok-length-format-retention-data) ; [OpusClip — Shorts retention](https://www.opus.pro/blog/ideal-youtube-shorts-length-format-retention) ; [OpusClip — LinkedIn video](https://www.opus.pro/blog/ideal-linkedin-video-length-format-for-retention) ; [We3Media — 2025 video lengths](https://we3media.us/best-video-length-for-instagram-reels-tiktok-and-youtube-shorts-in-2025/).

**R7. Une vidéo = une plateforme cible. Le repost brut cross-plateforme (avec watermark TikTok sur Reels, format paysage sur Shorts, ton Instagram sur TikTok) est détecté et sous-distribué. Adapter au minimum : hook, sous-titres, ratio, son.**
Sources : [Creatively Squared — Top 10 brand mistakes on TikTok](https://www.creativelysquared.com/article/top-10-mistakes-brands-are-making-on-tiktok) ; [Bazaarvoice — New rules of TikTok](https://www.bazaarvoice.com/updates/the-new-rules-of-tiktok-what-brands-and-creators-need-to-know/).

### D. Erreurs « corporate » qui tuent la portée

**R8. Le rendu publicitaire léché est contre-productif en organique : les utilisateurs (et l'algorithme, via leurs signaux de skip) rejettent le contenu poli/mis en scène. Produire « créateur » : smartphone, face caméra, lumière naturelle, montage cut rapide, imperfections assumées.**
Anti-patterns à bloquer au brief : logo animé en intro, voix off institutionnelle, plan produit sur fond studio en ouverture, discours « nous sommes fiers de », pack-shot sans humain.
Sources : [Creatively Squared — brand mistakes TikTok](https://www.creativelysquared.com/article/top-10-mistakes-brands-are-making-on-tiktok) ; [Adrenaline — TikTok brand audit 2025](https://www.adrenalinex.com/blog/tiktok-brand-audit-guide/).

**R9. TikTok 2025-2026 : divulgation du contenu commercial obligatoire. Tout contenu promotionnel (marque, produit, code promo, lien affilié, CTA « shop now ») doit être étiqueté via l'outil natif, sinon : notification sous 2-3 h, puis exclusion du feed For You si non corrigé sous 24 h.**
L'IA de TikTok détecte automatiquement mentions de marque, placements produit et hashtags promotionnels. Un agent qui scripte une vidéo promo doit inclure « activer le label contenu commercial » dans la checklist de publication.
Sources : [TikTok — Branded Content Policy](https://www.tiktok.com/legal/page/global/bc-policy/en) (source primaire) ; [TikTok Ads — Commercial content disclosure](https://ads.tiktok.com/help/article/about-the-commercial-content-disclosure-setting-for-advertisers) ; [Darkroom — TikTok 2026 policy update](https://www.darkroomagency.com/observatory/what-brands-need-to-know-about-tiktok-new-rules-2026).

**R10. Le CTA agressif en début/milieu de vidéo fait fuir : garder le CTA pour les 5 dernières secondes, une seule action demandée, formulée comme une suite logique du contenu (« la partie 2 est sur le profil », « le template est en commentaire ») plutôt qu'un impératif commercial.**
Sources : [Socialync — structure guide 2026](https://www.socialync.io/blog/short-form-video-structure-guide-2026) ; [Teleprompter — short-form strategy 2026](https://www.teleprompter.com/blog/short-form-video-strategy).

## 3. À intégrer dans les skills du template

| Skill | Intégration concrète |
|---|---|
| `video-editing` | Gabarit de script imposé : Hook 0-3 s / Value 4-15 s / Payoff / CTA 5 dernières secondes, avec beat toutes les 5-7 s (R2) ; règle de montage « couper 20-30 % du script » (R3) ; interdiction logo/jingle/plan d'installation en ouverture (R1, R8) ; durées cibles par plateforme dans le front-matter du brief (R6) ; une déclinaison par plateforme, pas de repost brut (R7). |
| `captions` / `embedded-captions` | Sous-titres incrustés obligatoires sur toute vidéo livrée, avec les chiffres sound-off en justification (R4) ; specs : zone sûre hors UI, 1-2 lignes, mots-clés en évidence (R5) ; test de validation « la vidéo se comprend-elle en muet ? » avant livraison. |
| `social-content` (scripts vidéo) | Le hook verbal est écrit en premier et validé seul (règle des 3 s, R1) ; payoff explicite exigé au brief — pas de payoff, pas de production (R2) ; CTA unique en fin, formulé en continuité (R10). |
| `brand-check` | Ajouter un axe « anti-corporate » pour la vidéo organique : détecter voix off institutionnelle, ouverture logo, pack-shot sans humain, ton « communiqué » (R8) ; checklist TikTok : label contenu commercial activé pour tout contenu promotionnel (R9). |
| `content-strategy` | Positionner la vidéo courte par objectif : TikTok 60-180 s pour l'autorité/éducation, Reels 15-30 s pour la portée, Shorts 25-40 s, LinkedIn < 90 s notoriété (R6) ; KPI de pilotage = taux de complétion et watch time, pas les vues brutes (R3). |
