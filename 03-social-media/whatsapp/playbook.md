## Cadence et rôle du canal

- Cadence maximale : {{CONTENT_CADENCE_WHATSAPP}} — WhatsApp est un canal intime : la sur-sollicitation coûte des désabonnements définitifs.
- Usage : alertes courtes et rappels d'événements uniquement — moins de 50 mots par message.

## Règles universelles

1. Un message = une information = un lien maximum.
2. Toujours actionnable : date, heure, lien d'inscription — pas de contenu éditorial long.
3. Respect strict de l'opt-in : uniquement les contacts qui ont explicitement accepté ce canal (voir `12-acquisition/conformite-rgpd.md` si le module acquisition est actif).
4. Pas d'emoji en rafale ; suivre la politique emoji de `01-brand/voice.md`.
